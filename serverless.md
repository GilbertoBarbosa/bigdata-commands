Install
```
npm install -g serverless
```

Create project
```
serverless create --template=<template> --path=<projectName>
```

Deploy
```
serverless deploy
```

Pubsub
```
npm install @google-cloud/pubsub
```

BigQuery
```
npm install @google-cloud/bigquery
```

Koa
```
npm install koa
npm install koa-bodyparser
```

Basic-auth
```
npm install basic-auth
```
