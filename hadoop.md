Version
```
hadoop version
```

Start/stop hadoop service
```
start-dfs.sh
stop-dfs.sh
```

Start/stop yarn service
```
start-yarn.sh
stop-yarn.sh
```

Check status Hadoop services
```
jps
```

List directory
```
hdfs dfs -ls <path>
```

Create directory
```
hdfs dfs -mkdir <directoryName>
```

Copy files
```
hdfs dfs -put <origin> <destination>
```
